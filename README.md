# freelancer-dashboard
A multi-page dashboard for a fictional freelance client using React
## Key Features
Responsive layout optimized for desktop and mobile devices.

Clean, modern interface designed for usability.

Overview page that contain stats about projects, earnings, recent activity, and calendar.

Projects page that contain organized project list in a table.

Profile page where the user can edit name, email, and profile picture.

Navigation from page to another using React Router without page reloads.

## Tech Stack
React + Vite - Recharts - React Router - React Calendar
